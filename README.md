# Sorts
### What is Sorts
>Sorts is a C++ test program to test some kinds of array-sorting algorithms. It also Works in system32, and it receives parameters from console.

### Algorithms
* Bubble Sort
* Selection Sort
* Insertion Sort
* Merge Sort
* Quick Sort
* Heap Sort
* Radix Sort
